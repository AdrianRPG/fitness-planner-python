# 🏋️ GYM: Start Your New Lifestyle

A Python-based fitness management application that helps users generate personalized workout routines and nutrition plans based on their personal data, health conditions, and fitness goals.

---

## 🎯 Objective

The goal of this project is to simulate a real-world fitness assistant system that:
- Collects user data through structured input
- Evaluates health and fitness conditions
- Generates personalized diet plans
- Recommends workout routines based on experience level

---

## 🧠 Features

- Interactive CLI (Command-Line Interface)
- Input validation and error handling
- Personalized calorie and macronutrient calculations
- Health condition assessment
- Fitness experience evaluation
- Custom workout routine generator

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook / Python Script
- Basic CLI (input/output handling)

---

## 📚 Skills Demonstrated

- Python programming fundamentals
- Conditional logic and control flow
- Function design and modular programming
- User input validation
- Data processing and calculations
- CLI-based application design

---

## ⚙️ How It Works

1. The user enters personal information (age, gender, height)
2. The program evaluates health conditions
3. The user provides fitness experience and lifestyle data
4. The system calculates:
   - Daily calorie needs
   - Macronutrient distribution (protein, carbs, fats)
5. Based on user goals, the program recommends:
   - Diet plan (bulking, cutting, recomposition)
   - Workout routine (beginner, intermediate, advanced)

---

## ▶️ How to Run

1. Open the notebook:
```bash
jupyter notebook gym-fitness-app.ipynb
```
2. Run all cells.
